The Pipeline
Config
Preset selection, tag registry initialization, path resolution. Supports generic, libar-generic, and DDD-ES-CQRS presets. src/config/
Scanner
File discovery, AST parsing, opt-in detection via @libar-docs marker. Supports TypeScript and Gherkin sources. src/scanner/
Extractor
Pattern extraction from TypeScript JSDoc annotations and Gherkin feature tags with shape resolution and cross-source merging. src/extractor/
Transformer
Builds MasterDataset with pre-computed views, dependency graphs, and O(1) lookups. The single read model for all consumers. src/generators/pipeline/
Codec
20 Zod codecs transform MasterDataset to RenderableDocument to Markdown. Progressive disclosure splits detailed vs compact output. src/renderable/

Get Started
01
Install the package
Add as a dev dependency. Works with pnpm, npm, and yarn. Requires Node.js ≥ 18 and ESM.
$ pnpm add -D @libar-dev/delivery-process
02
Annotate your sources
Add @docs markers to TypeScript JSDoc or Gherkin feature tags. Configure a preset that matches your architecture.
/** @docs */ /** * @docs-pattern MyPattern * @docs-status roadmap * @docs-uses SessionManager */
03
Generate documentation
Run the generator. Output includes pattern registries, roadmaps, remaining work, architecture diagrams, and decision records.
$ pnpm docs:all
